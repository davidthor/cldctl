# arcctl get environment

Get detailed information about an environment.

<Note>
Use `arcctl get env` as shorthand for `arcctl get environment`.
</Note>

## Synopsis

```bash
arcctl get environment <name> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<name>` | Name of the environment |

## Options

| Option | Description |
|--------|-------------|
| `-o, --output <format>` | Output format: `table`, `json`, `yaml` |
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |

## Examples

```bash
# Get environment details
arcctl get environment staging

# Using alias
arcctl get env staging

# Output as JSON
arcctl get environment staging -o json
```
